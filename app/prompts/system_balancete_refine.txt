CORREÇÕES A FAZER:

1. TIPO (A/D):
   - A = conta AGRUPADORA (totalizadora, soma de filhas)
   - D = conta de DETALHE (folha, sem filhas)
   - Verifique pela hierarquia: se o código tem subcódigos abaixo, é A.

   - Contas no último nível da hierarquia quase sempre são D.

2. NATUREZA (D/C):
   - Grupo 1 (Ativo): natureza D (Devedora)
   - Grupo 2 (Passivo + PL): natureza C (Credora)
   - Grupo 3 (Custos e Despesas): natureza D (Devedora)
   - Grupo 4 (Receitas): natureza C (Credora)
   - Retificadoras têm natureza INVERSA ao grupo pai.

3. SINAIS:
   - Converta "D" e "C" dos valores para sinais numéricos:
     - Se natureza do GRUPO é D: valor com "D" = positivo, valor com "C" = negativo
     - Se natureza do GRUPO é C: valor com "C" = positivo, valor com "D" = negativo
   - Remova as letras D/C dos valores numéricos.

4. HIERARQUIA:
   - is_totalizador: True para contas tipo A, False para tipo D.
   - nivel: conte os separadores "." na classificação + 1.

5. VALORES:
